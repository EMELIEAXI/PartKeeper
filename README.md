# Lagerhanteringssystem – Examensarbete

Detta projekt är ett lagerhanteringssystem utvecklat som examensarbete vid TUC Yrkeshögskola.  
Systemet består av en React/Vite-frontend, en ASP.NET Core-backend och en SQL-databas.

## Funktioner
- CRUD för reservdelar
- In- och utleveranser
- Användarautentisering (JWT)
- Rollhantering
- Realtidsuppdatering av lagerstatus
- Transaktioner med validering
